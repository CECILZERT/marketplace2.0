# mon-site-react

Structure minimal d'un projet full-stack (React frontend + Express/MongoDB backend).

## Démarrage rapide (backend)
1. cd backend
2. cp .env.example .env (ou éditer .env)
3. npm install
4. npm start

## Démarrage frontend
1. Installer les dépendances (Create React App attendu)
2. npm start

## Notes
- Le fichier `src/utils/api.js` pointe vers `http://localhost:5000/api`
- Cloudinary variables sont dans `backend/.env`
